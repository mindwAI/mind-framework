# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-13

### Added

- **install.sh**: POSIX sh installer with idempotent update/fresh/cancel and `--update` flag
- **setup.sh**: curl-bash one-liner wrapper (git clone + install + cleanup)
- **MIND.md**: 9-section agent index (~106 lines, auto-generated by install.sh)
- **AGENTS.md, CLAUDE.md, .cursorrules**: agent-agnostic shim files pointing to MIND.md
- **docs/customization.md**: guide for adding personas, rules, templates, pipeline steps
- **docs/writing-personas.md**: step-by-step guide with security-reviewer example
- **docs/migration-to-mindwai.md**: conceptual migration path for growing teams
- **examples/hello-world**: complete todo-app with 7 production-quality MIND artifacts (PRD, techspec, 3 tasks, review, QA)
- 6 mindwAI pain-point hooks in commands (execute-qa, validate-spec, create-tasks, execute-task, execute-review)

### Changed

- **MIND.md**: rewritten from 131-line manifesto to 106-line structured agent index
- **README.md**: rewritten for humans — quick start, comparison table, zero overlap with MIND.md
- **examples/hello-world/**: skeleton replaced with functional todo-app example

## [0.0.1] - 2026-04-04

### Added

- Initial release of the MIND Framework
- **Personas**: senior-developer, code-reviewer, qa-engineer, tech-writer, product-owner
- **Commands**: create-prd, clarify-prd, create-techspec, validate-spec, create-tasks, execute-task, execute-review, execute-qa
- **Rules**: code-standards, git-branching, tests
- **Templates**: prd-template, techspec-template, qa-checklist-template, feature-template
- MIND.md manifesto with philosophy, pipeline overview, and quick start
- Example project in examples/hello-world/
- Configuration schema in .mind/config.yaml
